=====================================
 TROUBLESHOOTING
=====================================

Q: MongoDB Connection Error?
A: Start MongoDB service first
   - Or use MongoDB Atlas (free cloud)
   - Update MONGODB_URI in .env

Q: Port 5000 already in use?
A: Change PORT in .env file
   Then: npm start

Q: Upload not working?
A: Check browser console (F12)
   Check server logs
   File might be too large

Q: Forgot .env file?
A: Copy .env.example to .env
   Edit with your settings
